# VSCode Drawing Board

一个简单的 VSCode 画板插件，让您可以直接在 VSCode 中绘画。

## 功能

- 基础绘画功能
- 多种颜色选择（黑色、红色、蓝色）
- 清除画布功能

## 使用方法

1. 按 `Ctrl+Shift+P` (Windows) 或 `Cmd+Shift+P` (Mac) 打开命令面板
2. 输入 "打开画板" 并选择该命令
3. 在打开的画板中进行绘画

## 快捷操作

- 点击颜色按钮切换画笔颜色
- 点击清除按钮清空画布
- 按住鼠标左键进行绘画

## 要求

- VSCode 1.80.0 或更高版本
